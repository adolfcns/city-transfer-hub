怎么创建令牌？（六步，1 分钟）
  1. 打开 github.com/settings/personal-access-tokens/new（Fine-grained token）
  2. Token name 随便填（如 city-hub）；Expiration 选最长（1 年）
  3. Repository access → 选 Only select repositories → 勾 city-transfer-hub
  4. Permissions → Repository permissions → 找到 Actions → 选 Read and write
  5. 点底部 Generate token
